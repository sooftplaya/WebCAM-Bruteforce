# WebCAM-Bruteforce
Web-CamBrute — инструмент для автоматического перебора логинов и паролей на IP-камерах.

Как использовать
Подготовка IP-адресов:

Откройте папку data.

В файл Ips.txt добавьте список IP-адресов, которые хотите проверить.

Можно использовать IP-диапазоны, найденные с помощью сканеров или вручную.

Добавление логинов и паролей:

Откройте папку login passwords.

В файле logins.txt укажите список логинов.

В файле passwords.txt укажите список паролей.

В файле payloads.txt можно настроить специальные запросы для авторизации.

Запуск скрипта:

Запустите start.bat (Windows) или start.sh (Linux).

Скрипт начнет сканирование, подбирая пароли к IP-камерам.

При успешном подборе логина и пароля информация отобразится в консоли.

Результаты работы:

Файл output.txt — найденные учетные данные (IP, логин, пароль).

Папка pictures — скриншоты с камер после успешного входа.
